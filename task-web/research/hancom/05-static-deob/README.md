# Static Deob Research

이 디렉터리는 Hancom Docs static deob 및 reverse-engineering 전용 코드를 보관한다.

- `src/`: static bundle mirror, module extraction, heuristic reconstruction, FR line parser
- `examples/`: live probe 및 artifact generation entry
- `tests/`: research helper 전용 test

메인 SDK 빌드, 기본 lint, 기본 test 범위에는 포함하지 않는다.
